# cs701-project-final

## Dependencies
1. http
2. formidable
3. express
4. swig
5. sqlite3

Install all dependencies in terminal in the same directory as the project

## Running the application
1. Migrate a new database in terminal
  
  knex migrate: make setup
  
  knex migrate: latest
2. run server.js in terminal 
  
  node server.js
